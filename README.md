#  PrepWise ‚Äì AI Resume Role Checker

PrepWise is a web application that allows users to upload their resume and check if they are ready for a selected job role.  
The system analyzes the resume and compares it against the chosen role requirements to give feedback.

---

##  Features

-  Upload Resume (PDF/DOC)
-  Select a Target Job Role
-  Resume Analysis
-  Skill & Readiness Feedback
-  GitHub profile link submission
